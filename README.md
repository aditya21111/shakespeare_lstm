## About The Project

### Next word prediction using LSTM


A next-word prediction model based on LSTM (Long Short-Term Memory) networks, 
trained on William Shakespeare's works sourced from the Gutenberg corpus. 
The model learns language patterns and generates context aware word predictions 
from a given text prompt.

## Getting Started


### Prerequisites

* pip
  ```sh
    pip install -r requirements.txt
  ```

### Run locally

1. Clone the repo
   ```sh
   git clone https://github.com/aditya21111/shakespeare_lstm.git
   ```
1. Install the required packages
   ```sh
   pip install -r requirements.txt
   ```
3. Run the streamlit app
   ```sh
   streamlit run app.py
   ```
